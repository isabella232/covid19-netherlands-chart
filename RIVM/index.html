Cloudflare Worker for RIVM - COVID-19 infections in the Netherlands

Introduction

Netherlands National Institute for Public Health and the Environment (RIVM) has an overview of information related to COVID-19 cases in the Netherlands: https://www.rivm.nl/nieuws/actuele-informatie-over-coronavirus .

The source of this information comes in the form of unstructured HTML.
I wanted to use this date for my COVID-19 chart but I did not want to manually update my data every day. To solve this, I create an API that parses the page and return JSON.

This is an HTML representation of the Javascript code in src/RIVM.js being executed.

It is mostly meant for development and testing purposes. For production use this Cloudflare Worker: https://covid-19.potherca.workers.dev/RIVM/

More information can be found at https://github.com/potherca-blog/covid19-netherlands-chart/
